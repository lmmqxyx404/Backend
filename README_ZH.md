# 项目简介
这是一个传统的restful风格后端工程项目，
我的目标是让这个项目能够对有志于使用rust开发web后端的新手提供有效的指引。
当然也可以理解为我个人的开发笔记

# 项目功能历史
## 0.1 验证码接口功能
完整实现了图形验证码接口功能

## 0.2 登录接口
### 登录方式暂时只考虑用户名密码登录
1. 密码后端拿到的是明文
2. 存储在数据库中的是密文
3. 采用对称加密
4. 后期会改进安全性
